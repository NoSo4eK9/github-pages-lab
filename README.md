# CyberLab — статичний сайт для GitHub Pages

Це простий HTML/CSS-проєкт для лабораторної роботи з розгортання статичного сайту через GitHub Pages.

## Структура

- `index.html` — головна сторінка
- `about.html` — сторінка про лабораторну роботу
- `projects.html` — додаткова сторінка з картками
- `styles.css` — файл стилів
- `.nojekyll` — службовий файл для GitHub Pages

## Як опублікувати

1. Створити public-репозиторій на GitHub.
2. Завантажити всі файли проєкту в репозиторій.
3. Відкрити Settings → Pages.
4. Обрати Source: Deploy from a branch.
5. Обрати Branch: main, Folder: /root.
6. Зберегти налаштування і відкрити URL сайту.
