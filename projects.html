Лабораторна робота №1

Розробка статичного веб-сайту CyberLab

Цей пробний сайт створений як частина лабораторної роботи з веб-розробки. Він демонструє основні принципи створення статичного сайту з використанням HTML, CSS та базової навігації. Сайт включає кілька сторінок, адаптивний дизайн та публікацію через GitHub Pages.

01

HTML-структура

Базова структура сайту з використанням семантичних тегів HTML5: header, main, footer, навігація між сторінками.

02

CSS-оформлення

Адаптивний дизайн з використанням CSS Grid та Flexbox для створення сучасного інтерфейсу.

03

GitHub Pages

Публікація статичного сайту через GitHub без окремого сервера та платного хостингу.

04

Версійний контроль

Використання Git для збереження історії змін, commit та push до віддаленого репозиторію.
