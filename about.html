Розгортання HTML-проєкту на GitHub Pages

Мета цього проєкту — показати базовий процес створення статичного сайту, завантаження його до репозиторію GitHub та публікації через GitHub Pages.

01

Створення файлів

Було створено кілька HTML-сторінок і файл стилів CSS.

02

Репозиторій GitHub

Проєкт завантажується у public-репозиторій, щоб GitHub Pages міг його опублікувати.

03

Активація Pages

У налаштуваннях репозиторію обирається гілка main та папка root.

04

Оновлення сайту

Після commit і push GitHub автоматично оновлює опубліковану версію сайту.
